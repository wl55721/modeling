# 测试说明

## 测试环境

- Python 3.7+
- 依赖：无额外依赖，使用项目内的模块

## 测试命令

### 运行所有算子测试

```bash
# 在项目根目录下运行
python tests/test_all_operators.py
```

### 运行特定算子测试

如果需要测试特定算子，可以修改 `test_all_operators.py` 文件，在 `test_all_operators` 函数中添加条件判断。

## 测试内容

测试脚本会验证以下内容：

1. **算子创建**：确保所有算子都能成功创建
2. **算子调用**：确保所有算子的 `__call__` 方法能正常工作
3. **动态输入构建**：确保所有算子的 `build_dynamic_input` 方法能正常工作
4. **属性检查**：确保所有算子都具有必要的属性

## 测试结果

测试运行后会输出每个算子的测试状态，包括成功或失败的详细信息。最终会显示总的测试结果，例如：

```
测试完成: 成功 37, 失败 0
```

## 注意事项

- 测试脚本使用了项目内的 `TensorBase` 和 `InputParam` 类，不需要额外安装依赖
- 测试脚本会自动检测所有注册的算子，并对每个算子进行测试
- 如果测试失败，会显示详细的错误信息，帮助定位问题
